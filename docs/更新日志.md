## 2026-07-14 · 管理后台与文件协作体验升级

### 新增

- 重构文件浏览首页与后台管理界面，统一玻璃质感、主题色、明暗模式、响应式布局、页面过渡和可访问交互。
- 文件浏览器支持存储源切换、URL 路径同步、面包屑、列表/画廊视图、文件详情统计、文件说明、文本编辑，以及图片、音视频、PDF 等内容预览。
- 新增多选与批量移动、复制、分享、直链等操作，并提供分享、直链、收藏的快捷管理入口。
- 分享支持单项或多项创建、访问密码、有效期、最大访问次数、浏览/下载统计、限制编辑与过期分享清理；公开分享可继续浏览嵌套目录。
- 存储源管理扩展为本地目录、S3 兼容存储、阿里云 OSS、腾讯云 COS 与 WebDAV，并加入字段校验、路径规则、排序、公开状态和启停配置。
- 新增主题管理，可选择多套主题预设并配置站点 Logo 与 favicon；品牌资源在公开页面和重新加载后保持一致。
- 新增公告管理，支持草稿/发布状态，并在公开文件页展示未读公告。
- 新增“关于 XFile”页面，读取 GitHub 仓库、作者、Stars、Forks、Releases、最近提交和 Markdown 文档；网络异常时可回退到缓存内容。

### 变更

- 后台导航按系统设置、链接与统计、规则与权限、项目重新分组，并补充折叠侧栏、全局搜索、移动端适配和快捷反馈。
- 显示设置、站点设置、访问控制、上传规则、用户规则和安全设置进一步模块化，保存后可立即应用并持久化。
- 分享、直链、收藏、存储源和文件列表页面改进桌面表格与移动卡片布局，补充搜索、空状态、加载状态和错误恢复。
- Toast 提示支持手动关闭；动态品牌、主题色和 favicon 会在窗口重新聚焦时同步。
- Docker CI 使用新版 Checkout、Go 与 Node Actions，前端依赖与锁文件同步更新。

### 接口与数据

- 统一并扩展 `/api/v1` 接口，新增文件详情、文件说明、批量文件操作、分享限制、公告、主题、品牌资源和 GitHub 关于信息等端点。
- SQLite 自动迁移文件说明、公告、分享访问上限与统计等数据结构，并补充存储层读写逻辑。
- 分享访问次数按根页面访问计数，同一会话内继续浏览目录或下载不会重复消耗配额。
- API 文档补充分享有效期与最大访问次数的创建、更新规则。

### 开源许可
